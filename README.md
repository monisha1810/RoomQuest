# RoomQuest 🏠

RoomQuest is a web-based PG accommodation discovery platform designed to help students and working professionals find suitable paying guest accommodations with ease.

## Features

* Search PG accommodations by city
* View detailed property information
* Explore amenities and rental details
* Responsive and user-friendly interface
* Interactive property listing system
* Interested property tracking

## Tech Stack

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3
* Bootstrap

### Backend

* PHP
* MySQL

## Installation

1. Clone the repository

```bash
git clone https://github.com/monisha1810/RoomQuest.git
```

2. Navigate to the project directory

```bash
cd RoomQuest/react-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open your browser and visit:

```text
http://localhost:3000
```
